LINUX HELP

  1. Navigation and Filesystem

    pwd

    Function: Print working directory.

    When to use: To know your current location in the filesystem.

    ls

    Function: List files and directories.

    Variations:

    ls -l (long format, details)

    ls -a (include hidden files)

    ls -lh (human-readable sizes)

    When to use: Exploring directory contents.

    cd

    Function: Change directory.

    Variations:

    cd /path/to/dir

    cd .. (up one level)

    cd ~ (to home directory)

    When to use: Moving around the filesystem.

    tree

    Function: Shows directory structure as a tree.

    When to use: For a visual overview of folders and subfolders.

  2. File Management

    touch

    Function: Create an empty file or update file timestamps.

    When to use: Quick file creation.

    cp

    Function: Copy files or directories.

    Variations:

    cp file1 file2

    cp -r dir1 dir2 (recursive copy)

    When to use: Duplicating files or directories.

    mv

    Function: Move or rename files/directories.

    When to use: To rename or reorganize files.

    rm

    Function: Delete files.

    Variations:

    rm file

    rm -r dir (recursive delete)

    rm -f file (force delete)

    When to use: Removing files or directories (Use caution).

    find

    Function: Search for files.

    Variations:

    find . -name "file.txt"

    find /path -type d -name "dir"

    When to use: Locating files based on name, type, or attributes.

  3. Viewing and Editing Files

    cat

    Function: Concatenate or view file contents.

    When to use: Quick file preview.

    less

    Function: Paginated file viewing.

    When to use: Viewing large files.

    head and tail

    Function: Show first/last lines of a file.

    Variations:

    head -n 20 file

    tail -f logfile (live updates)

    When to use: Checking beginnings or ends of files, logs.

    nano, vim, gedit

    Function: Text editors.

    When to use: Editing giles directly from terminal (choose based on preference)

  4. Permissions and Ownership

    chmod

    Function: Change file permissions.

    Variations:

    chmod 755 file

    chmod u+x script.sh

    chown

    Function: Change file owner/group.

    Variations:

    chown user file

    chown user:group file

    When to use: Adjust Ownership for users/services.

    umask

    Function: Sets default permission masks.

    When to use: Define default permissions for new files.

  5. Processes and System Monitoring

    ps

    Function: Show running processes.

    Variations:

    ps aux (all processes with details)

    When to use: Checking process status.

    top / htop

    Function: Live system monitoring.

    When to use: Tracking CPU, memory usage, and processes.

    kill

    Function: Terminate processes.

    Variations:

    kill [file]

    kill -9 [file] (force kill)

    jogs, bg, fg

    Function: Manage background/foreground jobs.

    When to use: Controlling multitasking in the shell.

  6. Disk and Filesystem Management

    df

    Function: Show disk space usage.

    Variations:

    df -h (human readable)

    When to use: Checking free/used space.

    du

    Function: Show file/directory space usage.

    Variations:

    du -sh dir

    When to use: Checking how much space a directory takes.

    mount / unmount

    Function: Mount/unmount filesystems.

    When to use: Managing external drives, partitions.

  7. Networking

    ping

    Function: Test connectivity to a host.

    When to use: Troubleshooting network reachability.

    ifconfig / ip

    Function: Show or configure network interfaces.

    When to use: Checking or assigning IP addresses.

    curl

    Function: Transfer data from/to URLs.

    Variations:

    curl [url]

    curl -0 [file]

    When to use: Downloading files, API testing.

    wget

    Function: Downloading files from the internet.

    Variations:

    wget [link]

    wget -c [link] (resume download)

    When to use: Fetching files over HTTP/FTP.

    ssh

    Function: Secure shell login to remote machines.

    Variations:

    ssh user@host

    ssh -p 2222 user@host

    When to use: Remote administration.

    scp

    Securly copy files over SSH.

    Variations:

    scp [file] user@host:/path

    scp -r dir user@host:/path

    When to use: File transfer between systems.

  8. Archiving and Compression

    tar

    Function: Archive files.

    Variations:

    tar -cvf archive.tar files

    tar -xvf archive.tar

    tar -czvf archive.tar.gz files (with compression)

    When to use: Bundling files/directories.

    When to use: Bundling files/directories.

    gzip / gunzip

    Function: Compress/decompress files.

    When to use: Compressing single files.

    zip / unzip

    Function: Compress/uncompress zip archives.

    When to use: Handling zip files.

  9. Package Management

    apt update && apt upgrade (update packages)

    apt install package

    apt remove package

    When to use: Installing, updating, removing software.

  10. User and System Management

    whoami

    Function: Show current user

    id

    Shows user ID and group info.

    adduser / useradd

    Function: Create new user accounts.

    passwd

    Function: Change user password.

    sudo

    Function: Run commands with elevated privileges.

    When to use: For administrative tasks.

    shutdown / reboot

    Function: Power off or restart system.

    Variations:

    shutdown -h now

    reboot

  11. Helpful Shortcuts

    . (current directory)

    .. (parent directory)

    ~ (home directory)

    * (wildcard)

    > (redirect output)

    >> (append output)

    | (pipe output between)

    && (run multiple commands if first succeeds)

    ; (run multiple commands sequentially)

    Ctrl+C (stop process)

    Ctrl+Z (suspend process)

    fg (resume in foreground)

    bg (resume in background)
