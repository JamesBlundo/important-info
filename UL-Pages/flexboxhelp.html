  1. Container Basics

    display

    Function: Turns an element into a flex container.

    Common Variations:

    display: flex; (block-level container)

    display: inline-flex; (inline-level container)

    When to use: Whenever you need flexible alignment and layout for child elements.

  2. Main Axis Control

    flex-direction

    Function: Defines the main axis (row vs. column).

    Common Variations:

    row (default — left to right)

    row-reverse (right to left)

    column (top to bottom)

    column-reverse (bottom to top)

    When to use: To change the flow direction of flex items.

    flex-wrap

    Function: Controls whether flex items wrap onto multiple lines.

    Common Variations:

    nowrap (default — all items on one line)

    wrap (items wrap onto multiple lines)

    wrap-reverse (wrap in reverse order)

    When to use: To allow items to break into multiple rows/columns.

    flex-flow

    Function: Shorthand for flex-direction + flex-wrap.

    Common Variations:

    flex-flow: row wrap;

    When to use: Compactly define direction and wrapping.

  3. Main Axis Alignment

    justify-content

    Function: Aligns flex items along the main axis.

    Common Variations:

    flex-start (default — start of axis)

    flex-end (end of axis)

    center (centered)

    space-between (equal space between items)

    space-around (equal space around items)

    space-evenly (equal spacing everywhere)

    When to use: To control horizontal alignment in rows or vertical alignment in columns.

  4. Cross Axis Alignment

    align-items

    Function: Aligns items along the cross axis.

    Common Variations:

    stretch (default — stretch to fill)

    flex-start (align at cross-axis start)

    flex-end (align at cross-axis end)

    center (centered)

    baseline (align text baselines)

    When to use: To vertically align items in a row or horizontally align items in a column.

    align-content

    Function: Aligns multiple lines of flex items along the cross axis (only applies if wrapping).

    Common Variations:

    stretch (default — fill container)

    flex-start

    flex-end

    center

    space-between

    space-around

    When to use: To control spacing between wrapped rows or columns.

  5. Individual Item Control

    align-self

    Function: Overrides align-items for a specific flex item.

    Common Variations:

    auto (default, inherits from parent)

    flex-start

    flex-end

    center

    baseline

    stretch

    When to use: To align one item differently from others.

    order

    Function: Controls the order of items.

    Default: 0

    When to use: To visually reorder items without changing HTML structure.

    flex

    Function: Shorthand for flex-grow, flex-shrink, and flex-basis.

    Common Variations:

    flex: 1 1 auto;

    flex: 0 0 200px; (fixed basis, no grow/shrink)

    When to use: Quick control over how items grow/shrink.

    flex-grow

    Function: Defines how much an item grows relative to siblings.

    Example: flex-grow: 2; (item grows twice as fast)

    flex-shrink

    Function: Defines how much an item shrinks when space is tight.

    Example: flex-shrink: 0; (item won’t shrink)

    flex-basis

    Function: Initial size of item before growing/shrinking.

    Example: flex-basis: 200px;

  6. Gap & Spacing

    gap

    Function: Controls spacing between flex items.

    Common Variations:

    gap: 20px;

    row-gap: 10px; column-gap: 20px;

    When to use: Cleaner alternative to margins for spacing items.

  7. Comparison: When to Use Which

    justify-content vs align-items: Use justify-content for main-axis alignment, align-items for cross-axis.

    align-items vs align-content: Use align-items for single-line containers, align-content for multi-line wrapping.

    order vs HTML rearranging: Use order for quick reordering, but prefer semantic HTML changes when possible.

    gap vs margins: Use gap for consistent spacing; margins for individual element control.
